# Spring Boot User CRUD API

## Features
- Create User
- Get User by ID
- Get All Users
- Update User
- Delete User

## Tech Stack
- Java
- Spring Boot
- MySQL

## API Endpoints
POST /Users/create  
GET /Users/getusers/{id}  
GET /Users/getalluser  
PUT /Users/update/{id}  
DELETE /Users/delete/{id}
